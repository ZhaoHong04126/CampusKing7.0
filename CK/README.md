# 👑 校園王 (CampusKing) - 數位學園

![Version](https://img.shields.io/badge/Version-9.2%20Dashboard-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Web%20%7C%20PWA-success.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

**校園王 (CampusKing)** 是一款專為大學生打造的「全方位校園生活儀表板」。
告別散落各處的筆記與試算表，我們將課表、成績、記帳、行事曆與學習計畫完美整合在一個現代化、高質感的雲端 SaaS 介面中。

## ✨ 系統亮點 (v9.2 全新架構)

在最新的 v9.2 版本中，我們迎來了史詩級的介面革新！捨棄了傳統的 App 網格設計，全面升級為**專業級雙欄儀表板 (Dashboard)**，提供更直覺、更流暢的操作體驗。

### 📊 核心管理模組
* **🏠 資訊儀表板 (Dashboard)：** 總覽當前學期狀態與今日任務速覽。
* **📅 智慧課表 (Schedule)：** 支援單日/週課表切換、自訂節次時間、課程色彩標記，更支援「一鍵匯出課表圖片」。
* **💯 成績與學分 (Grade Manager)：** 自動計算 GPA、追蹤畢業學分達成率、視覺化歷年成績趨勢圖。
* **🎒 作業與小考 (Homework)：** 集中管理各科作業繳交期限與得分。
* **💰 學期記帳 (Accounting)：** 專為學生設計的收支追蹤，包含圓餅圖與折線圖收支分析。
* **🗓️ 專屬行事曆 (Calendar)：** 整合校園活動與個人行程，支援月曆與清單雙視圖。

### 🧰 實用小工具
* **🧮 配分筆記：** 記錄各科教授的計分方式與目標分數。
* **📝 快速記事：** 靈感與待辦事項隨手記。
* **🎰 幸運籤筒：** 解決「午餐吃什麼」的千古難題。
* **💝 紀念日：** 倒數重要日子，不再忘記。
* **📚 學習目標：** 量化你的讀書計畫進度。

### ⚙️ 系統特性
* **雙頁面架構：** 獨立的入口落地頁 (`index.html`) 與主系統 (`app.html`)，載入更輕快。
* **響應式設計 (RWD)：** 電腦版顯示專業側邊欄，手機版自動轉換為滑出式抽屜選單。
* **雲端同步：** 整合 Firebase Auth 與 Firestore，資料即時備份不遺失。
* **本地備份還原：** 支援將大學回憶打包匯出為 JSON 檔，隨時無縫還原。

## 🛠️ 技術架構

* **前端核心：** HTML5, CSS3, Vanilla JavaScript (無框架，極致輕量)
* **後端與資料庫：** Google Firebase (Authentication, Cloud Firestore)
* **圖表與視覺化：** Chart.js
* **畫面截圖工具：** html2canvas

## 🚀 快速開始

1. 將本專案複製到本地端。
2. 確保你在 `js/firebase.js` 中填入了你專屬的 Firebase 設定金鑰。
3. 透過 VS Code 的 Live Server 或部署至任何靜態網頁伺服器（如 GitHub Pages, Vercel）。
4. 開啟 `index.html` 即可看到專屬的數位學園入口！

# 📅 校園王 (CampusKing) v9.1

> 大學生活很精彩，但總是被早八的課表、期中的死線和月底的荷包追著跑？
> 告別零散的備忘錄與截圖，這是我為學生（和我自己）量身打造的「數位手帳」—— 用一個 Web App，搞定所有的校園日常！

## 🚀 為什麼同學都愛用校園王？

校園王不只是一個工具，它是採用「類手機桌面 (Web OS)」架構開發的超級助理。我們把大學生活所需的一切，濃縮在這個極致純粹的系統裡。

### 🌟 專屬你的「純淨模組化首頁」 (v9.1 全新升級)
- **漸進式探索與自訂首頁**：初次登入時，透過直覺的雙排勾選介面，自由決定首頁要顯示哪些核心功能。
- **設定控制中心**：將「更多工具箱」與「系統資訊」從首頁徹底抽離，打造極致乾淨的版面。隨時可至「⚙️ 個人設定」中一鍵管理首頁模組，或直接啟動隱藏的 App。

### 📅 不再跑錯教室的「智慧課表」
- **自訂你的作息**：每節課幾點下課自己定！完美支援不同大樓的通勤時間，純 24 小時制顯示更直覺。
- **視覺化管理**：為不同課程標上專屬的粉嫩色彩，連堂課會自動合併，一眼就能看懂今天的行程。
- **一鍵存桌布**：排好課表後，點擊「📷 存成圖片」就能無縫變成手機桌布！

### 💯 拯救期末焦慮的「成績與作業」
- **學分進度條**：GPA 幫你算好算滿，距離畢業還有幾學分？看上方的進度條就對了。
- **期末配分筆記**：「這科平時佔 30%、期中佔 40%...」把教授的規則記下來，過關目標超清晰。
- **死線救星**：條列式管理待繳作業與考試，做完一鍵打勾，享受把事情清空的快感。

### 💰 擺脫月底吃土的「學期記帳」
- **極速滑動輸入**：像用 Excel 一樣流暢，一口氣把整天的花費橫向記下來。
- **甜甜圈圖表**：生活費到底花去哪？一秒看出餐費、網購的比例，幫你抓出漏財兇手。
- **誠實面對「糊塗帳」**：發現錢包餘額對不上？系統會發出靈魂拷問，強制將想不起來的差額記為警示灰色的「糊塗帳」，督促你養成隨手記帳的好習慣！

### 🛠️ 點綴校園的「貼心小工具」
- **🎰 幸運籤筒**：「中午吃學餐還是麥當勞？」讓命運的轉盤幫選擇困難的你決定。
- **💝 紀念日倒數**：在一起幾天了？距離連假還有多久？通通幫你算好。
- **🗓️ 連續色塊行事曆**：跨日活動（如迎新、宿營）用絕美色塊連接，媲美原生 iOS 體驗。

### 🛡️ 默默保護你的「底層魔法」
- **防誤觸上鎖**：手機放口袋亂滑？閒置一分鐘自動進入「🔒 唯讀模式」，保護你的資料不被搞砸。
- **雙重備份**：換手機也不怕！資料自動雲端同步，還能一鍵下載「專屬大學回憶包」存在自己電腦裡。
- **無痕極簡導覽**：移除畫面上多餘的個資顯示與裝飾，讓注意力完全集中在你的校園生活。

## 💻 我使用的技術架構 (Tech Stack)

整個專案採用原生前端技術打造，輕量且極致流暢，免下載 App 就能直接加入手機桌面。

- **前端 (Frontend)**: HTML5, CSS3, JavaScript (Vanilla ES6+)
- **後端 (Backend)**: Google Firebase (Firestore, Authentication)
- **圖表與工具 (Tools)**: `Chart.js` (數據視覺化), `html2canvas` (課表截圖匯出)
