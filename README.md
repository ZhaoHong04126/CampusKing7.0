# 📅 校園王 (CampusKing) v9.0

> 大學生活很精彩，但總是被早八的課表、期中的死線和月底的荷包追著跑？
> 告別零散的備忘錄與截圖，這是我為學生（和我自己）量身打造的「數位手帳」—— 用一個 Web App，搞定所有的校園日常！

## 🚀 為什麼同學都愛用校園王？

校園王不只是一個工具，它是採用「類手機桌面 (Web OS)」架構開發的超級助理。我們把大學生活所需的一切，濃縮在這個流暢的系統裡。

### 🌟 專屬你的「模組化動態首頁」
- **漸進式探索 (Onboarding)**：初次使用不再眼花撩亂！透過簡單的學期目標問卷，系統會自動挑選最適合你的 App 組合。
- **聰明收納工具箱**：將次要功能收納至底部的「🧰 更多工具箱」中，降低視覺與認知負荷，把介面的主導權完美交還給使用者。

### 📅 不再跑錯教室的「智慧課表」
- **自訂你的作息 (v8.9 新增)**：每節課幾點下課自己定！完美支援不同大樓的通勤時間，純 24 小時制顯示更直覺。
- **視覺化管理**：為不同課程標上專屬的粉嫩色彩，連堂課會自動合併，一眼就能看懂今天的行程。
- **一鍵存桌布**：排好課表後，點擊「📷 存成圖片」就能無縫變成手機桌布！

### 💯 拯救期末焦慮的「成績與作業」
- **學分進度條**：GPA 幫你算好算滿，距離畢業還有幾學分？看上方的進度條就對了。
- **期末配分筆記**：「這科平時佔 30%、期中佔 40%...」把教授的規則記下來，過關目標超清晰。
- **死線救星**：條列式管理待繳作業與考試，做完一鍵打勾，享受把事情清空的快感。

### 💰 擺脫月底吃土的「學期記帳」
- **極速滑動輸入**：像用 Excel 一樣流暢，一口氣把整天的花費橫向記下來。
- **甜甜圈圖表**：生活費到底花去哪？一秒看出餐費、網購的比例，幫你抓出漏財兇手。
- **誠實面對「糊塗帳」**：發現錢包餘額對不上？系統會發出靈魂拷問，強制將想不起來的差額記為警示灰色的「糊塗帳」，督促你養成隨手記帳的好習慣！

### 🛠️ 點綴校園的「貼心小工具」
- **🎰 幸運籤筒**：「中午吃學餐還是麥當勞？」讓命運的轉盤幫選擇困難的你決定。
- **💝 紀念日倒數**：在一起幾天了？距離連假還有多久？通通幫你算好。
- **🗓️ 連續色塊行事曆**：跨日活動（如迎新、宿營）用絕美色塊連接，媲美原生 iOS 體驗。
- **📚 讀書進度條**：段考複習進度視覺化，讀到哪裡一目了然。

### 🛡️ 默默保護你的「底層魔法」
- **防誤觸上鎖**：手機放口袋亂滑？閒置一分鐘自動進入「🔒 唯讀模式」，保護你的資料不被搞砸。
- **雙重備份**：換手機也不怕！資料除了自動雲端同步，還能一鍵下載「專屬大學回憶包」，把青春穩穩地存在自己電腦裡。
- **深夜護眼**：完美支援深色模式，半夜趕報告看課表也不刺眼。
- **多學期支援**：支援建立無限多個學期（如 114-1, 114-2），陪伴你度過完整的四年。

## 💻 我使用的技術架構 (Tech Stack)

整個專案採用原生前端技術打造，輕量且極致流暢，免下載 App 就能直接加入手機桌面。

- **前端 (Frontend)**: HTML5, CSS3, JavaScript (Vanilla ES6+)
- **後端 (Backend)**: Google Firebase (Firestore, Authentication)
- **圖表與工具 (Tools)**: `Chart.js` (數據視覺化), `html2canvas` (課表截圖匯出)

```text
│
├── css/                # 樣式表 (Stylesheets)
│   ├── base.css        # 全域變數、Reset、字體設定

│   ├── components.css  # [UI元件] 按鈕、卡片、Modal 視窗

│   ├── auth.css        # 登入表單樣式
│   ├── dashboard.css   # 桌面與主要 APP 共用樣式
│   ├── settings.css    # 設定頁面樣式
│   └── calendar.css    # 行事曆專屬樣式
│
└── js/                 # 腳本邏輯 (Scripts)
    ├── [核心系統]
    ├── firebase.js     # Firebase 初始化與連線設定
    ├── auth.js         # 使用者身分驗證邏輯

    ├── main.js         # 系統啟動入口、狀態監聽

    ├── data.js         # 資料庫存取 (CRUD) 封裝

    ├── [功能 APP]

    ├── notes.js        # 快述記事本
    ├── anniversary.js  # 紀念日倒數

```
